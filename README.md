# WinForms CSV Loader

A Windows Forms application built with .NET 9 that loads and displays CSV files with advanced features.

## Features

- 📂 Load CSV files with file picker dialog
- 📊 Display data in a DataGridView
- ⚡ Async loading with progress indicator
- 🔍 Handles large datasets efficiently
- ✅ Comprehensive error handling
- 🎨 Responsive UI with proper anchoring

## Technologies

- .NET 9
- C# 13.0
- Windows Forms
- CsvHelper library

## Requirements

- Windows OS
- .NET 9 Runtime
